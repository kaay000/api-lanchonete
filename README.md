# API Lanchonete

Projeto desenvolvido com FastAPI para gerenciamento de pedidos de uma lanchonete.

## Tecnologias

- Python
- FastAPI
- Pytest

## Funcionalidades

- Cancelamento de pedidos
- Observações
- Prioridade
- Fila de preparo

## Executar

```bash
uvicorn main:app --reload
```

## Testes

```bash
pytest -q
```
